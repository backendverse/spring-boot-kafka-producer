## ⚙️ Project Setup

1. ## 🐳 Run with Docker Compose (Recommended)

If you don’t want to install Kafka manually, you can use **Docker Compose**.

1. Start containers:

   ```bash
   docker compose up -d
   ```

2. Check running containers:

   ```bash
   docker ps
   ```
